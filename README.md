# Recipe_Page
## Description
A responsive "Simple Omelette Recipe" page layout, built with semantic HTML5 and custom CSS styling. The project helps users with preparing Omelette recipe, with the needed ingredients, instructions, and also the suitable prepation time. Some nutrients are shown for those who follow a healthy system. In addition, the design is clear for eye with beautiful and integrated colors.
